### 🚀 Proyectos Destacados

#### 🏢 [ProyGreen ERP](https://tipkev.com/proygreen/login.php)
Sistema ERP integral construido para el sector agroindustrial. Unifica módulos de Recursos Humanos, Logística y Vigilancia bajo una arquitectura de seguridad estricta.
* **Características:** Arquitectura de roles y permisos (RBAC), gestión de asistencia y planillas, módulo de Registro de Accidentes e Inspecciones (SST).
* **Stack:** `PHP` `MySQL` `JavaScript` `AJAX`

#### 🔗 [QRKat](https://tipkev.com/qrkat/)
Generador avanzado de códigos QR dinámicos y estáticos.
* **Características:** Permite a los usuarios crear códigos personalizados para enlaces, vCards, WhatsApp y correos electrónicos con una interfaz intuitiva.
* **Stack:** `PHP` `JavaScript` `CSS`

#### 🌱 [Web Corporativa Greenvic](https://tipkev.com/Greenvic/)
Sitio web corporativo e informativo destacando la producción agrícola.
* **Características:** Identidad corporativa moderna, catálogo de productos y acceso directo al portal interno de trabajadores.
* **Stack:** `JavaScript` `HTML5` `Tailwind CSS`
